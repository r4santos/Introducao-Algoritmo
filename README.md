# Introdução a Algoritmos

Repositório acadêmico da disciplina **Introdução a Algoritmos**.

Este espaço reúne atividades, exercícios e práticas desenvolvidas ao longo da matéria. Mesmo já tendo experiência prévia em programação, estou refazendo a disciplina por ser componente obrigatório da universidade, com foco em consolidar fundamentos, organizar soluções e registrar minha evolução acadêmica.

## Objetivo do repositório

- Registrar as atividades propostas em aula.
- Praticar lógica de programação com diferentes níveis de dificuldade.
- Reforçar conceitos essenciais para disciplinas futuras.
- Manter um histórico organizado de exercícios resolvidos.

## Estrutura do projeto

O repositório está organizado por listas e etapas de atividades:

- **Arquivos da base de exercícios**
  - exemplos introdutórios com entrada, processamento e saída
  - cálculos simples, média, IMC, conversões e operações matemáticas

- **AP03**
  - atividades com foco em estruturas iniciais de decisão e cálculos aplicados

- **AP04**
  - lista de exercícios de estruturas condicionais
  - decisões com if/elif/else, operadores lógicos e condicionais aninhadas

- **AP05**
  - lista de exercícios com estruturas de repetição usando while
  - contadores, acumuladores, validação de dados e menus interativos

## Conteúdos praticados

Ao longo das atividades, os principais tópicos trabalhados são:

- lógica de programação
- variáveis e tipos de dados
- entrada e saída de dados
- operadores aritméticos e relacionais
- estruturas condicionais
- estruturas de repetição
- validação de entrada
- organização de algoritmos no fluxo:
  - Entrada -> Processamento -> Saída

## Observações acadêmicas

- Este repositório tem finalidade educacional.
- As soluções priorizam clareza e aderência ao enunciado das atividades.
- Sempre que possível, os exercícios são organizados de forma progressiva para facilitar revisão de conteúdo.
